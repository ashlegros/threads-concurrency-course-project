# threads-concurrency-course-project
An upload of an OOP project demonstrating the threads and concurrency concept.

Name: Ashbel Legros
Course: Software Development I
CRN: CEN 3024C - 31032

Description: The project simulates a pushup battle between two contenders. It uses concepts such as threads and functions such as thread sleeping to simulated rest periods in between sets of pushups. When you run main, two contenders should start their battle and the program exits when a contender reaches a certain amount (set to 200) of pushups.
